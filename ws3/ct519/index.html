สรุปความรู้ในการดำเนินการข้อสอบนี้

การดำเนินการข้อสอบนี้มีขั้นตอนหลัก ๆ ที่ต้องปฏิบัติ ดังนี้:

1. การตั้งค่า AWS และการจัดการ EC2 Instances

สมัครและตั้งค่า AWS Account: เริ่มต้นโดยการสมัคร AWS Account และเข้าสู่ระบบ AWS Management Console

สร้าง EC2 Instances: สร้าง EC2 Instances จำนวน 4 เครื่อง (1 สำหรับ Nginx Proxy และ 3 สำหรับ Web Servers)

2. การติดตั้ง Docker และ Docker Compose
การตั้งค่า Nginx Proxy สำหรับ Load Balancing

สร้าง Docker Compose File สำหรับ Nginx: กำหนดการตั้งค่า Nginx ให้ทำงานเป็น Proxy และ Load Balancer โดยใช้ไฟล์ docker-compose.yml และ nginx.conf

การกำหนดค่า Nginx เพื่อ Load Balance: ใช้การตั้งค่า upstream และ proxy_pass เพื่อส่งการร้องขอไปยัง Web Servers ทั้ง 3 ตัว

การตั้งค่า Web Servers และเพิ่มเนื้อหา

สร้าง Docker Compose File สำหรับแต่ละ Web Server: สร้างไฟล์ docker-compose.yml สำหรับ Web1, Web2, และ Web3

การขอ SSL Certificate

ใช้ Certbot: เพื่อขอ SSL Certificate จาก Let's Encrypt สำหรับการใช้งาน HTTPS บนโดเมนที่กำหนด

การแจ้งหมายเลข IP

แจ้ง Elastic IP: ที่ได้รับจาก AWS ใน Comment ของ Assignment ตามที่กำหนด

บทเรียนที่ได้รับ

การจัดการ AWS และ EC2:

เรียนรู้วิธีการสร้างและตั้งค่า EC2 Instances รวมถึงการจัดการ Security Groups และ Elastic IP

การติดตั้งและใช้งาน Docker:

ได้ฝึกฝนการติดตั้งและใช้งาน Docker และ Docker Compose เพื่อสร้างบริการที่ต้องการ

การตั้งค่า Nginx Proxy และ Load Balancing:

ได้เรียนรู้การตั้งค่า Nginx ให้ทำงานเป็น Proxy และ Load Balancer เพื่อกระจายการร้องขอไปยัง Web Servers หลายตัว

การใช้ Certbot เพื่อขอ SSL Certificate

เข้าใจขั้นตอนและการใช้ Certbot เพื่อขอ SSL Certificate จาก Let's Encrypt

การออกแบบและสร้างเว็บไซต์

เรียนรู้วิธีการสร้างเว็บไซต์เบื้องต้นด้วย HTML และการตั้งค่า Docker Compose สำหรับ Web Servers

การดำเนินการข้อสอบนี้ช่วยให้นักศึกษาได้รับประสบการณ์ในการใช้เทคโนโลยีที่สำคัญในด้านการพัฒนาและการจัดการระบบ ซึ่งจะมีประโยชน์ในการทำงานจริงในอนาคต
